# MyShop E-Commerce Website

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Features

- User Registration
- User Login
- Product Listing
- Search Products
- Shopping Cart
- Checkout
- Order Management
- MongoDB Database

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

Open:

```text
frontend/index.html
```

in browser.

## API Endpoints

### Users

POST /api/users/register

POST /api/users/login

### Products

GET /api/products

POST /api/products

### Orders

GET /api/orders

POST /api/orders
